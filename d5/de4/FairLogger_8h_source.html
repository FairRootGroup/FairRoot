72 // Definition of the different verbosity levels of the output
73 // With the level HIGH the output contains a time stamp, the location from
74 // which the message commes (File, Function and Line) toegether with the
75 // log level and the mesage to be printed.
76 // The verbosity level MEDIUM has the above information without the time stamp
77 // The LOW verbosity level prints only the log level together with the message
78 enum FairLogVerbosityLevel {verbosityHIGH, verbosityMEDIUM, verbosityLOW};
